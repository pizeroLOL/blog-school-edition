## 术语

照顾一些可能没听说过这些东西的人。

### LaTeX

一种排版排版系统，这篇文章多指Latex语句在Markdown里的应用，比如说写数学函数什么的。

类似html，只不过尖括号变为各种反斜杠，类似这样：

```LaTeX
$\frac{x}{y}$
```

显示效果长这样：$\frac{x}{y}$

就很好玩，也很好用，不过学习成本略高，还有一些控制大小的语句啥的。

pandoc没有控制上下标的时候为直接拿这玩意写。

### Markdown

一种文档语言，就是 txt 的 pro max 版，缩写为 md，在编程领域用途广泛（特指国外，国内全 tm csdn 收费，人都麻了）。

有一些很好玩的功能，详见[测试页][test-page]，语法见测试页的 [markdown 文档][test-page-md]。

这篇文章也是用 md 写的——当然你实际看到的时候也可能是我用 pandoc 转换后的网页。

规整，方便，即开即用，即使未经过转换也有非常高的美观度，哪怕你直接用记事本打开都非常好看，更何况上手难度并不高。

[test-page-md]: blogs/1/Readme.md
